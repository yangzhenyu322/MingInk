# 技术选型
开发框架：SpringBoot、SpringCloud、OpenFeign  
缓存：Redis分布式缓存 + 本地缓存  
数据库：MySQL、MongoDB  
网关：Gateway + Sentinel  
分布式事务：Seata、事务消息  
持久层：MyBatis、MyBatis-Plus  
注册中心：Nacos  
消息队列：RocketMQ、Kafka  
网络通信：Netty，WebSocket  
文件存储：MinIO、OSS  
日志可视化治理：ELK  
监控：Prometheus，Grafana  
单元测试：Junit  
压力测试：JMeter
